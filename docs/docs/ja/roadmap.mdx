      <strong>部分的な完成によって証明</strong>されたとき——1 つの Layer 3 レシピ、

      残作業は再キューイングされる。これにより、いつまでも閉じない「未完フェーズ」が
      じわじわと累積していく事態からプロジェクトを守っている。
    </p>
